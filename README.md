Короткий опис розробленої програми

Програма реалізована мовою C# на платформі .NET 8 у вигляді консольного застосунку. Вона виконує три операції лінійної алгебри на основі процедури звичайних жорданових виключень (ЗЖВ): пошук оберненої матриці, обчислення рангу матриці та розв'язання системи лінійних алгебраїчних рівнянь другим способом.

Структура програми

Програма розбита на 8 класів, кожен з яких відповідає за свою зону відповідальності.

Program — точка входу програми, містить головне меню та виклик відповідних солверів залежно від вибору користувача. Меню з'являється повторно після завершення кожної операції.

JordanElimination — реалізує саму процедуру ЗЖВ. Це ядро всіх обчислень. Метод Eliminate виконує один крок жорданових виключень: розв'язувальний елемент замінюється оберненим, елементи розв'язувального рядка діляться на нього, елементи стовпця діляться зі зміною знаку, а решта перераховується за формулою b_ij = a_ij - a_is * a_rj / a_rr.

InverseMatrixSolver — пошук оберненої матриці. Послідовно виконує n процедур ЗЖВ з розв'язувальними елементами по головній діагоналі. Після всіх кроків матриця перетворюється на обернену.

RankCalculator — обчислення рангу. Перебирає діагональні елементи, і якщо елемент ненульовий — виконує ЗЖВ та збільшує лічильник рангу. Нульові елементи пропускаються. Ранг дорівнює кількості успішних кроків.

SlaeSolver — розв'язання СЛАР другим способом. Формує розширену матрицю [A | -B], після чого на кожному кроці виконує ЗЖВ з розв'язувальним елементом у першому стовпці, змінює знак елементів розв'язувального рядка і викреслює перший стовпець. Після n кроків залишається стовпець з розв'язками.

Protocol — клас для запису протоколу обчислень. Виводить у консоль вхідні дані, інформацію про кожен крок (номер, розв'язувальний елемент), проміжні та кінцеві матриці. Вхідні матриці виводяться у чистому форматі, а проміжні — з точністю до 2 знаків після коми.

MatrixInput — зчитування даних з консолі. Підтримує введення цілих та дробових чисел, як через крапку так і через кому.

TestRunner — автоматичне виконання тестових прикладів на захардкожених даних без участі користувача. Містить по 2-3 приклади для кожної операції.

Вхідні та вихідні дані

Для пошуку оберненої матриці користувач вводить розмірність n та квадратну матрицю A. Програма виводить обернену матрицю C разом з протоколом кожного кроку ЗЖВ.

Для обчислення рангу вводяться розміри n та m і прямокутна матриця A. Програма виводить ранг матриці та протокол обчислень з поміткою про пропущені нульові елементи.

Для розв'язання СЛАР вводяться розмірність n, матриця коефіцієнтів A та вектор вільних членів B. Програма виводить вектор розв'язків X та повний протокол: переписану систему [A | -B] і результат кожного кроку ЗЖВ.

Пункт меню "Тестові обчислення" виконує всі три операції на заздалегідь підготовлених даних, включаючи варіант завдання A = [[3,5,1],[-2,2,-3],[1,3,-2]], B = [1,7,4].

Елементи рядка матриці вводяться через пробіл. Протокол обчислень виводиться безпосередньо в консоль.
